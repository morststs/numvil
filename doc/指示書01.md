# 概要
計算のパズルゲーム。

- ４つのペイン「パーツ」・「式」・「結果」・「題」に分かれる。
- 最初に「題」に数値が表示される。
- 「パーツ」には「0123456789()!+-/*^」がそれぞれ単体にドラッグできる。
- 「式」にパーツをドロップして配置して計算が可能な場合は「結果」に計算結果を表示する。
- 「結果」と「題」の数値が合えばクリア。
- クリアしたら次の「題」を更新
- パーツはそれぞれ１つの式には１回しか使えない
- 二つの数値をあわせて１桁以上の数は表せれない。例えば１と０を合わせて１０にすることはできない。
- 題は必ず１つ以上の回答の式がある前提とする。
- 降参すると回答がでる。

# アウトプット
- Windowsでのexe
- Androidアプリ
- Webでの静的コンテンツ
